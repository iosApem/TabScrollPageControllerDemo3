# TabScrollPageControllerDemo3
关键字: tab scroll scrollView page 
滚动标签页视图控制器

#优势
1.可以制定 标签滚动视图 的位置和大小, 以及父容器

2.可以制定 页滚动视图 的位置和大小, 以及父容器

3.支持加入控制器/简单的视图
